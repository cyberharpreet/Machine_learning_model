# Solid Waste Prediction using Machine Learning

This project predicts future solid waste generation using past yearly waste data.

The project uses Linear Regression to find the trend in waste generation and predict waste for any future year entered by the user.

The dataset contains year-wise municipal solid waste data of Punjab districts measured in tons/year.

Technologies Used:
- Python
- Pandas
- Scikit-learn

Example:
User enters year → Model predicts future waste generation.
